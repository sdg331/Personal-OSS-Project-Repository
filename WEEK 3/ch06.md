- **가볍지만 강력한** 소스 코드 편집기
- **크로스 플랫폼 지원**

- JavaScript

## VS Code 설치

### 다운로드

#### 공식 사이트
- **URL**: https://code.visualstudio.com/
- **다운로드 페이지**: https://code.visualstudio.com/download
- **파일명**: `VSCodeSetup-x64-1.XX.X.exe`

#### 1. 유저 버전 (User Installer)
- 현재 사용자만 설치 가능한 버전

#### 2. 시스템 버전 (System Installer) ✅ **권장**
- 해당 컴퓨터의 모든 사람이 사용 가능
- **시스템 설치 기본 폴더**: `C:\Program Files\Microsoft VS Code`

#### Windows (Windows 8, 10, 11)

- System Installer: x64, x86, Arm64
- .zip: x64, x86, Arm64
- CLI: x64, x86, Arm64

#### Linux
- **Debian, Ubuntu**: .deb (x64, Arm32, Arm64)
- **Red Hat, Fedora, SUSE**: .rpm (x64, Arm32, Arm64)
- .tar.gz: x64, Arm32, Arm64
- Snap Store
- CLI: x64, Arm32, Arm64

- ☑ 바탕 화면에 바로가기 만들기(D)

**장점**: 탐색기에서 파일 또는 디렉토리에서 바로 'vscode로 열기' 메뉴를 사용할 수 있어 편리[1]

#### 사용 방법
1. Windows 탐색기에서 원하는 폴더로 이동
2. 폴더 내 빈 공간에서 **우클릭**

## 리눅스 기초 명령어

### 폴더 관련 명령

#### pwd (Print Working Directory)
```bash
$ pwd
/c/[git tutorial]
```
- 현재 작업 중인 폴더 경로 표시
- "현재 어디에 있는가?"를 확인

#### cd (Change Directory)
```bash
$ cd basic          # basic 폴더로 이동
$ cd ..             # 상위 폴더로 이동
$ cd ~              # 홈 디렉토리로 이동
$ cd /c/git         # 절대 경로로 이동
```
- 디렉토리 이동 명령어

#### mkdir (Make Directory)
```bash
$ mkdir dname       # dname 폴더 생성
$ mkdir -p a/b/c    # 중첩 폴더 생성
```
- 새 디렉토리 생성

#### ls (List)
```bash
$ ls                # 파일 목록
$ ls -l             # 파일 상세 정보
$ ls -a             # 숨김 파일 포함
$ ls -al            # 상세 정보 + 숨김 파일
```
- 파일 또는 폴더 목록 표시

### ls 명령어 옵션 상세

| 옵션 | 설명 |
|------|------|
| `-l` | 파일의 상세 정보 표시 |
| `-a` | 숨김 파일(`.`으로 시작하는 파일) 표시 |
| `-al` | 상세 정보 + 숨김 파일 (조합) |
| `-t` | 파일들을 생성 시간 순(제일 최신 것부터)으로 표시 |
| `-rt` | 파일들을 생성 시간 순(제일 오래된 것부터)으로 표시 |
| `-F` | 파일 유형을 나타내는 문자를 파일명 끝에 표시 |

**파일 유형 표시**:
- `/`: 디렉터리
- `*`: 실행 파일
- `@`: 링크

## 파일 관련 명령어

### 파일 생성

#### touch (파일 생성)
```bash
$ touch fname       # 빈 파일 fname 생성
$ touch a.txt b.txt # 여러 파일 동시 생성
```
- 빈 파일 생성
- 기존 파일이 있으면 수정 시간만 변경

#### echo (문자열 출력)
```bash
$ echo git bash                    # 단순 문자열 출력
$ echo 'print()'                   # 괄호 포함 시 따옴표 사용
$ echo "Hello World"               # 큰따옴표도 가능
```
- 표준 출력으로 결과 표시
- 이후 문자열을 컴퓨터 터미널에 출력(display)
- 단순한 문자열은 따옴표 생략 가능
- 괄호 등 특수문자가 있는 경우 작은 또는 큰 따옴표 사용

### 파일 내용 보기

#### cat (Concatenate)
```bash
$ cat fname         # 파일 내용 출력
$ cat file1 file2   # 여러 파일 내용 출력
```
- 파일의 내용을 화면에 출력
- 여러 파일을 연결하여 출력 가능

#### cat 활용 (파이프와 함께)
```bash
$ cat file1 file2 | more    # 페이지별로 출력
$ cat file1 file2 | head    # 처음 10줄만 출력
$ cat file1 file2 | tail    # 끝 10줄만 출력
```

**파이프 (|)의 의미**:
- 앞 명령의 표준 출력을 뒤 명령의 표준 입력으로 처리
- 명령어 연결

### 파일 복사 및 이동

#### cp (Copy)
```bash
$ cp a b            # 파일 a를 b로 복사
$ cp file1 dir/     # file1을 dir 폴더로 복사
$ cp -r dir1 dir2   # 디렉토리 복사
```
- 파일 또는 디렉토리 복사

#### mv (Move)
```bash
$ mv f1 f2          # 파일 f1을 f2로 이름 변경
$ mv file dir/      # file을 dir 폴더로 이동
```
- 파일 이름 변경 또는 이동

### 파일 및 폴더 삭제

#### rm (Remove)
```bash
$ rm fname          # 파일 fname 삭제
$ rm -f fname       # 강제로 파일 삭제
$ rm -rf dname      # 폴더 및 하위 모든 내용 삭제
```

**옵션 설명**:
- `-f`: 강제로 파일이나 디렉토리를 삭제 (확인 없이)
- `-r`: 디렉토리 내부의 모든 내용을 재귀적으로 삭제

**주의**: `.git` 폴더 삭제 시 Git 저장소 기능 상실
```bash
$ rm -rf .git       # Git 저장소 삭제 (주의!)
```

## 리다이렉트 (Redirection)

### 리다이렉트 연산자

#### > (출력 리다이렉트 - 덮어쓰기)
```bash
$ echo aaa > a.txt      # 파일 a.txt에 "aaa" 저장 (기존 내용 삭제)
$ cat file1 > file2     # file1 내용을 file2에 저장
```
- 화면의 출력 결과를 파일로 저장
- **기존에 있는 파일 내용을 지우고 저장**

#### >> (출력 리다이렉트 - 추가)
```bash
$ echo bbb >> a.txt     # 파일 a.txt에 "bbb" 추가
$ cat file4 >> file3    # file3에 file4의 내용 추가
```
- **기존 파일 내용 뒤에 덧붙여서 저장**

#### < (입력 리다이렉트)
```bash
$ cat < file1           # file1의 결과 출력
```
- 파일의 데이터를 명령에 입력

### 리다이렉트 활용 사례

```bash
# 1. 파일 생성 및 내용 작성
$ echo aaa > a.txt              # a.txt 생성, "aaa" 저장

# 2. 파일에 내용 추가
$ echo bbb >> a.txt             # a.txt에 "bbb" 추가

# 3. 여러 파일 병합
$ cat file1 file2 > file3       # file1, file2 내용을 file3에 저장

# 4. 파일 내용 추가
$ cat file4 >> file3            # file3에 file4 내용 추가

# 5. 파일 내용 출력
$ cat < file1                   # file1 내용 출력
```

## 파이프 (Pipe)

### 파이프 연산자 |

#### 개념
- **앞 명령의 표준 출력**을 **뒤 명령의 표준 입력**으로 처리
- 여러 명령어를 연결하여 복잡한 작업 수행

### 파이프 활용 예시

```bash
# 페이지별 출력
$ cat file1 file2 | more

# 처음 10줄만 출력
$ cat file1 file2 | head

# 끝 10줄만 출력
$ cat file1 file2 | tail

# 파일 목록을 시간순 정렬 후 페이지별 출력
$ ls -lt | more

# 특정 단어 검색
$ cat file.txt | grep "keyword"
```

### 리눅스 명령어 (Git CLI에서 함께 사용)

```bash
# 폴더 확인 및 이동
$ pwd               # 현재 위치 확인
$ ls -al            # 파일 목록 상세 조회

# 파일 생성 및 편집
$ touch a.txt       # 빈 파일 생성
$ echo aaa > a.txt  # 파일에 내용 작성

# 파일 내용 확인
$ cat a.txt         # 파일 내용 출력

# 파일 복사 및 이동
$ cp a.txt b.txt    # 파일 복사
$ mv a.txt b.txt    # 파일 이동/이름 변경

# 파일 삭제
$ rm a.txt          # 파일 삭제
$ rm -rf folder     # 폴더 삭제
```
